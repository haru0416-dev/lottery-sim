# セキュリティポリシー

## 脆弱性の報告

このプロジェクトは**バックエンドを持たない静的なクライアントサイドアプリ**です
（ユーザーデータの収集・保存・送信はありません。設定はブラウザの localStorage と
URL ハッシュにのみ保持されます）。

セキュリティ上の問題を見つけた場合は、**公開 Issue ではなく**以下で非公開に報告してください。

- GitHub の **Security → Report a vulnerability**（Private vulnerability reporting）

再現手順・影響範囲を添えていただけると助かります。可能な範囲で速やかに対応します。

## 対象範囲の目安

- XSS / CSP 回避 / 依存ライブラリの既知脆弱性 / ヘッダー設定の不備 など
- スコープ外: 物理的な確率・賞金データの正確性（仕様。アプリ内「データの前提と注意」参照）、
  自分のブラウザの localStorage 改変など攻撃者自身にしか影響しないもの。
